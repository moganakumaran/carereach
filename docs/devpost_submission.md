## Elevator pitch (2–3 sentences, if a longer summary is needed)
```
CareReach turns 10,000 messy Indian healthcare records into maternal-health deployment decisions a
planner can trust. Instead of one "desert score," it computes two independent signals — the care gap
and how much verified evidence backs it — so a real desert is never confused with a region we're
simply data-poor about. Ask by voice in Hindi, Bengali or Tamil, drill into the verbatim evidence
behind every claim, and save the plan to Lakebase. Built end-to-end on Databricks.
```
